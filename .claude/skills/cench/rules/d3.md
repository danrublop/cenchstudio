# D3 Scene Rules

---

## Output format

Output raw JSON only. No markdown fences. No explanation:

```json
{
  "styles": "<CSS string ‚Äî no <style> tags>",
  "sceneCode": "<JavaScript using D3 v7 global>",
  "suggestedData": { ... }
}
```

The template injects `styles` into a `<style>` block, `sceneCode` into a `<script>`, and pre-populates `DATA` from `suggestedData`.

---

## Globals available in sceneCode

- `d3` ‚Äî D3 v7 (loaded from CDN)
- `DATA` ‚Äî pre-filled from `suggestedData`
- `WIDTH` = 1920
- `HEIGHT` = 1080

---

## SVG setup

Always create the SVG this way:

```js
const svg = d3.select('#chart')
  .append('svg')
  .attr('viewBox', '0 0 1920 1080')
  .attr('width', '100%')
  .attr('height', '100%');
```

---

## Animation patterns

All enters must transition:

```js
.transition()
.duration(800)
.ease(d3.easeCubicOut)
.delay((d, i) => i * 100)
```

- Elements start at opacity 0, transition to 1
- Bars start at height 0, transition to full height (transform from bottom)
- Use `d3.easeCubicOut` for most transitions; `d3.easeElasticOut` for emphasis

---

## Font sizes

- Title: 48px bold
- Axis labels: 20px
- Data labels: 18px
- Annotations: 16px

---

## Chart types and patterns

**Bar chart:** use `scaleBand` for x, `scaleLinear` for y. Add axes with `axisBottom`/`axisLeft`. Animate bars with height 0 ‚Üí full.

**Line chart:** use `line()` generator. Animate with `stroke-dasharray` / `stroke-dashoffset` trick.

**Pie/donut:** use `arc()` + `pie()`. Animate with `attrTween` on `d`.

**Scatter:** circles fade in with staggered delay.

---

## suggestedData

Provide a realistic dataset matching the prompt topic. Structure it so `DATA` is directly usable in `sceneCode` without transformation. Use real-world plausible numbers.

---

## Colors

Use the palette in your styles: `#181818, #121212, #e84545, #151515, #f0ece0`

- Bars/lines: `#e84545` (accent) or `#f0ece0` (light)
- Backgrounds: `#181818`
- Text: `#f0ece0`
- Axes/gridlines: `#333333` or `rgba(240,236,224,0.2)`

---

## HTML template

The D3 template is injected by the rendering system. For the file write, use:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <style>
    * { margin: 0; padding: 0; box-sizing: border-box; }
    html, body { width: 100%; height: 100vh; overflow: hidden; background: BG_COLOR_HERE; }
    #chart { width: 100%; height: 100%; }
    STYLES_HERE
  </style>
</head>
<body>
  <div id="chart"></div>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/d3/7.9.0/d3.min.js"></script>
  <script>
    const DATA = D3_DATA_HERE;
    const WIDTH = 1920, HEIGHT = 1080;
    window.__pause  = () => {};
    window.__resume = () => {};
    SCENE_CODE_HERE
  </script>
</body>
</html>
```

Replace `BG_COLOR_HERE`, `STYLES_HERE` (raw CSS, no `<style>` tags), `D3_DATA_HERE` (JSON-serialized suggestedData), and `SCENE_CODE_HERE` (the sceneCode string).
